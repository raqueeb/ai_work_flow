# নেটওয়ার্ক মনিটরিং

এই গাইডে লোকাল LLM‑ভিত্তিক নেটওয়ার্ক মনিটরিং ডেমো দেখানো হয়েছে, যা পিং, ট্রেসরুট এবং DNS লুকআপের ফলাফল বিশ্লেষণ করে দ্রুত নেটওয়ার্ক স্বাস্থ্য চেক করে।

## ডেমো চালানো

```bash
python gemma-4-e4b-network_monitor.py
```

অথবা Streamlit UI:

```bash
streamlit run gemma-4-e4b-network_monitor.py
```

## ফিচার

- **পিং টেস্ট** – হোস্টের রেসপন্স টাইম ও প্যাকেট লস পরিমাপ।  
- **ট্রেসরুট** – রাউটার হপ‑গুলোর তালিকা ও লেটেন্সি।  
- **DNS লুকআপ** – ডোমেইন রেজোলিউশন যাচাই।  
- **LLM বিশ্লেষণ** – সমস্যা সনাক্ত করে সুপারিশ প্রদান।

## ব্যবহার

1. হোস্টনেম বা IP ইনপুট দিন।  
2. "Run Diagnostics" বাটন ক্লিক করুন।  
3. LLM থেকে স্বাস্থ্য রিপোর্ট দেখুন।

*নেটওয়ার্ক ট্রাবলশুটিং দ্রুত করতে এই টুল ব্যবহার করুন।*
