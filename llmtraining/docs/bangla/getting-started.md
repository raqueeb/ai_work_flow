# শুরু (লিগেসি)

এই গাইডটি লিঙ্ক৩‑এর লোকাল LLM প্রশিক্ষণ ও ডিপ্লয়মেন্টের জন্য প্রাথমিক ধাপগুলো বর্ণনা করে। এটি পুরনো Qwen 2.5‑1.5B মডেল ভিত্তিক সেটআপ।

## পূর্বশর্ত

- Python 3.10+  
- LM Studio (মডেল রান করার জন্য)  
- 8 GB+ RAM (Qwen‑এর জন্য)

## দ্রুত শুরু

1. **LM Studio ইনস্টল করুন** – [lmstudio.ai](https://lmstudio.ai) থেকে ডাউনলোড করুন।  
2. **মডেল ডাউনলোড করুন** – LM Studio‑এ `qwen2.5-coder-1.5b-instruct` সার্চ করে ইনস্টল করুন।  
3. **মডেল চালু করুন** – "Start Server" বাটন ক্লিক করে `http://localhost:1234` তে API চালু করুন।  
4. **প্রথম ডেমো চালান** – টার্মিনালে:

```bash
python app-reasoning1.py
```

- **ডেমো এক্সপ্লোর** – [qwen-demos.md](qwen-demos.md) পেজে সব ডেমোর তালিকা আছে।
