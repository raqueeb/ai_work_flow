# Qwen 2.5 দিয়ে শুরু করা

এই গাইডটি **Qwen 2.5‑1.5B** মডেল ব্যবহার করে লোকাল LLM ইনফারেন্স সেটআপের ধাপগুলো বর্ণনা করে। Windows, macOS এবং Linux-এ কাজ করবে।

## পূর্বশর্ত

- **Python 3.11+** – `python` আপনার PATH-এ থাকতে হবে।  
- **Git** – রেপোজিটরি ক্লোন করার জন্য (ঐচ্ছিক)।  
- **LM Studio** – https://lmstudio.ai/ থেকে ডাউনলোড ও ইনস্টল করুন।

## ইনস্টলেশন

```bash
# রেপোজিটরি ক্লোন (যদি না থাকে)
git clone https://github.com/raqueeb/link3_ai_cases.git
cd classifier-app

# ডিপেন্ডেন্সি ইনস্টল
pip install streamlit requests playwright
```

## LM Studio-এ Qwen 2.5‑1.5B সেটআপ

1. LM Studio চালু করুন।  
2. বাম সাইডবারে **Discover** (দাঁড়ি চিহ্ন) ক্লিক করুন।  
3. সার্চ বারে `Qwen2.5-1.5B-Instruct` লিখে মডেলটি খুঁজে বের করুন।  
4. **GGUF** সংস্করণ (Q4_K_M রিকোয়েস্টেড) ডাউনলোড করুন।  
5. **Models** ট্যাবে গিয়ে মডেলটি সিলেক্ট করে **Load** করুন।  
6. LM Studio স্বয়ংক্রিয়ভাবে `http://localhost:1234/v1` এ একটি লোকাল OpenAI‑কম্প্যাটিবল সার্ভার চালু করবে।

## Streamlit অ্যাপ চালানো

```bash

```

ব্রাউজার `http://localhost:8501` এ খুলবে, যেখানে টিকিট পেস্ট করে **Analyze Ticket** বাটন চাপলে ফলাফল দেখা যাবে।
