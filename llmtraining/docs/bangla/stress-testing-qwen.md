# স্ট্রেস টেস্টিং (Qwen 2.5)

এই গাইডে **Qwen 2.5‑1.5B** মডেল ব্যবহার করে স্ট্রেস‑টেস্ট চালানোর পদ্ধতি বর্ণনা করা হয়েছে। ২০০টি সিন্থেটিক টিকিটের মাধ্যমে মডেলের সঠিকতা, লেটেন্সি এবং দুর্বলতা বিশ্লেষণ করা যায়।

## চালানোর ধাপ

```bash
python llm_stress_test_class.py
```

## ফলাফল বিশ্লেষণ

- **সঠিকতা** – টিকিটের সঠিক কোডের শতাংশ।  
- **লেটেন্সি** – গড় রেসপন্স টাইম (সেকেন্ডে)।  
- **ব্রেকডাউন** – কোন ক্যাটেগরি (কীওয়ার্ড, CoT, ফ্যালব্যাক) সবচেয়ে বেশি ত্রুটি দিচ্ছে।
